# 📦 Django Delivery Service
#hi
Бэкенд-сервис для международной доставки. Позволяет регистрировать посылки, рассчитывать стоимость доставки с учётом веса, типа груза и актуального курса USD/RUB. Пользователи изолированы по сессиям.

## ✨ Описание

Сервис решает задачу расчёта стоимости доставки для международных отправлений.  
Пользователь (без авторизации) регистрирует посылку, указывая её параметры.  
Фоновая задача каждые 5 минут рассчитывает стоимость для всех новых посылок по формуле:

**Стоимость = (вес × 0.5 + стоимость_в_долларах × 0.01) × курс_USD/RUB**

Курс доллара кешируется в Redis для снижения нагрузки на внешнее API.

## 🚀 Быстрый старт

Запустите все сервисы (Django, PostgreSQL, Redis, Celery) одной командой:


docker-compose up --build
После запуска API будет доступно по адресу:
👉 http://localhost:8000/swagger/
